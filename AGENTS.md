# Macodoro

A MacOS-native pomodoro application.

## User requirements

- The application has to be written in Swift, so it's MacOS-native.
- The application has to be simple, yet modern.
- It has to follow the current styling best practices of modern applications.
- It should let the user set: working time, resting time, big resting time, iterations of working-resting until big resting time.
- It should show alerts.
- It should have a way to change the theme and provide different sets of colors.
- It should have a settings window.
- It should be able to be shown on the MacOS settings bar, as an icon.
- It should have a hovering mode, in which the clock should be shown over all the currently open applications.
- It should have a setting to set transparency.
- It should have different kinds of alerts.
- It should be installable through homebrew.
- It should be scriptable, meaning that the user should be able to open, close and change the settings through the terminal.
- It should save the settings locally, on the user's machine.
